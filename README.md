# Interactive Brokers Web API

Interactive Brokers Web API plugin for bt_api plugin for `bt_api`.

## Features

- Stock trading support\n- IB Web API interface

## Installation

```bash
pip install bt_api_ib_web
```

## Usage

```python
from bt_api_bt_api_ib_web import ib_web
# Use the exchange
```

## Architecture

The package follows the standard bt_api plugin structure with feeds, containers, and gateway adapters.
